# 📚 課程筆記範本

## 欄位設計

| 欄位名稱 | 類型 | 說明 |
|---------|------|------|
| 科目別 | Relation → Collection | 連結到科目分類的 Collection |
| 科目 | Text | 課程名稱（如：線性代數） |
| 章節 | Number | 第幾章 |
| 每章重點 | Rich Text | 核心概念、公式、定理 |
| 每章練習 | Text | 練習題記錄 |
| 每章典範題 | Text | 經典題目 + 解答 |
| 小提示 | Text | 考試技巧、容易犯錯的地方 |
| 進度 | Select | 未開始 / 進行中 / 已完成 |

---

## 📋 使用方式

1. **建立 Collection**「科目別」→ 例如：數學、物理、程式
2. **建立 Type**「課程筆記」並套用上述欄位
3. **建立範本** 
   - 物件右上角 ⋯ → Save as template
   - 命名「課程筆記範本」

## 🔧 快速建立 Steps

1. 打開 Anytype
2. 建立新 Type：「課程筆記」
3. 新增欄位：
   - 科目別 (Relation)
   - 科目 (Text)
   - 章節 (Number)
   - 每章重點 (Rich Text)
   - 每章練習 (Text)
   - 每章典範題 (Text)
   - 小提示 (Text)
   - 進度 (Select)
4. 建立第一個筆記物件
5. 右上角 ⋯ → Save as template
6. 命名「課程筆記範本」
