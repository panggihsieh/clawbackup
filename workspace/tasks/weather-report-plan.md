# 📋 天氣報告自動化 - 專案規劃

## 🎯 任務目標
定時自動回報 **嘉義市** 與 **嘉義縣梅山鄉** 的天氣資訊

---

## 🔄 執行流程

```
┌─────────────────────────────────────────┐
│           天氣報告自動化流程             │
├─────────────────────────────────────────┤
│  ┌─────────────┐    ┌─────────────┐    │
│  │  1. Cron   │───▶│  2. 查詢    │    │
│  │   觸發     │    │   天氣 API   │    │
│  │ (06:00/18) │    │  (wttr.in)  │    │
│  └─────────────┘    └──────┬──────┘    │
│                            ▼           │
│  ┌─────────────┐    ┌─────────────┐    │
│  │  4. Discord │◀───│  3. 生成   │    │
│  │    發送    │    │    報告    │    │
│  └─────────────┘    └─────────────┘    │
└─────────────────────────────────────────┘
```

### 步驟說明

| 步驟 | 動作 | 說明 |
|:----:|------|------|
| 1 | Cron 觸發 | 每日 06:00 與 18:00 準時觸發 |
| 2 | 查詢天氣 | 调用 wttr.in API 獲取天氣數據 |
| 3 | 生成報告 | 格式化為中文輸出（溫度/體感/風速/濕度）|
| 4 | Discord 發送 | 發送到指定頻道 |
